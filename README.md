# Bilibili-Portrait
**让 B 站视频适配你的竖屏，拯救颈椎的副屏神器。**()

## 作用

- 可自由拖动当前播放的bilibili视频。
  - 如拖动到屏幕下半部分的位置，**便于眼睛平视观看。**
- 🔓 自由拖拽： 可自由拖拽视频位置。随心移动至屏幕中下部，实现平视观看。
- 📐 随心缩放： 自由调整播放器尺寸，完美契合竖向显示器宽度，不再受页面布局束缚。
- 🧘 颈椎友好： 平视观看。
